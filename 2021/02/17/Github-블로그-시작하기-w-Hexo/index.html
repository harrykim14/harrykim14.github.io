Github 블로그 시작하기 w/ Hexo

Feb 17, 2021

새롭게 깃헙 블로그를 만들어보았다 w/ hexo

1. 블로그를 시작하게 된 이유

  • 첫째, 매일 커밋을 하고 있지만 정작 눈에 띄게 남는 것이 보이지 않아서
  • 둘째, 앞으로 코딩 테스트를 준비함에 있어 알고리즘은 똑부러지게 정리를 해야 할 것 같아서
  • 셋째, 책을 읽고 정리하는데 하나의 Readme.md로는 부족했기 때문에 (이 부분은 리팩토링 2판 페이지를 참조)

2. hexo를 택한 이유

  • Javascript가 익숙한 언어이고 cli도 잘 되어있으면서 무엇보다 hexo 공식문서가 어느정도 한글을 지원하고 있어 사용하기에 편할 것이라는 생각에 선택하게 됨

3. 앞으로 올릴 것들 및 올 한해 포부 등

  • 자료구조 및 알고리즘 관련 내용을 매일같이 공부하고 공부한 만큼 업데이트를 할 예정
  • 공부하는 도중에 둘러보는 UI/UX 디자인 페이지와 새 기술 등에 관련된 내용도 정리할 예정
  • 올해에는 원하는 회사에 취업하기!
